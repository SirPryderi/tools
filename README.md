# :wrench: SirPryderi/tools
A set of command line utilities and tools I use to make my life easier. 

# :package: Install

```bash
git clone git@github.com:SirPryderi/tools.git "$HOME/tools" && \
echo "source $HOME/tools/tools.sh" >> "$HOME/.bashrc" && \
source "$HOME/.bashrc" && \
sirpryderi
```
